## 步骤

### clone仓库

```powershell
git clone http://github.com/Soulter/AstrBot
cd AstrBot
```



### 虚拟/安装依赖

虚拟环境

```powershell
python3 -m venv ./venv
```

安装依赖

```powershell
venv\Scripts\activate
python -m pip install -r requirements.txt -i https://mirrors.tuna.tsinghua.edu.cn/pypi/web/simple
```

### 执行

```c
python main.py
```
