### 提交

直接CV

```c

```

### 5.提交所有实验

根据我的实验，其实你在lab0就可以提交lab1，lab2，lab3…等所有实验，只要你每次提交之前把分支就改为对应的就可以。

提交lab0：

```bash
git checkout lab0;
git add .; git commit -m "lab"; git push
```

提交lab1：

```c
git checkout lab1;
git add .; git commit -m "lab"; git push
```

提交lab2：

```c
git checkout lab2;
git add .; git commit -m "lab"; git push
```
